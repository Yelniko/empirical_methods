# Lab_4
1. Назвіть мету регресійного аналізу

Мета регресійного аналізу — це дослідження залежності між змінними та побудова моделі, яка дозволяє:
- описати зв’язок між залежною та незалежними змінними;
- оцінити вплив факторів;
- робити прогнози значень;
- знаходити важливі змінні, що впливають на результат.

2. Наведіть коротку характеристику видів регресійного аналізу

- Лінійна регресія
  Найпростіша модель. Показує лінійну залежність: y = ax + b.

- Множинна лінійна регресія
  Використовує кілька змінних X для прогнозу Y.

- Поліноміальна регресія
  Використовує нелінійні залежності (x², x³ тощо).

- Ridge регресія (L2)
  Додає штраф за великі коефіцієнти, зменшує перенавчання.

- Lasso регресія (L1)
  Може зануляти деякі коефіцієнти, допомагає відбору ознак.

- ElasticNet
  Комбінація L1 і L2 регуляризації.


3. У чому відмінність класів Lasso, Ridge та ElasticNet від класів LassoCV, RidgeCV
та ElasticNetCV бібліотеки sklearn?

- без CV → параметри задає користувач
- з CV → параметри підбирає модель автоматично

# Lab_7

1. Які ще методи кластеризації існують крім KMeans?

Окрім методу KMeans існує багато інших алгоритмів кластеризації, які використовуються залежно від типу даних та поставленої задачі.

1. Hierarchical Clustering (Ієрархічна кластеризація).
- Будує дерево кластерів (дендрограму).
  - агломеративна (об’єднання об’єктів знизу вверх)
  - дивізивна (розбиття зверху вниз)
- Добре підходить для аналізу структури даних.

2. DBSCAN
- Групує об’єкти за щільністю розташування.
- Добре знаходить кластери складної форми.
- Може визначати шумові точки та викиди.

3. Mean Shift
- Шукає області найбільшої щільності даних.
- Не потребує заздалегідь вказувати кількість кластерів.

4. Gaussian Mixture Model (GMM)
- Вважає, що дані складаються із суміші нормальних розподілів.
- Дає ймовірність належності до кластерів.

5. Spectral Clustering
- Використовує графову структуру зв’язків між об’єктами.
- Добре працює для складних даних.

6. Birch
- Ефективний для великих наборів даних.
- Поступово будує дерево кластерів.

7. Affinity Propagation
- Сам визначає кількість кластерів.
- Визначає центральні точки (екземпляри).

2. Що обчислює параметр inertia_ побудованої кластеризації?

Параметр inertia_ у моделі KMeans показує:
Суму квадратів відстаней від кожного об’єкта до центру свого кластера.
Тобто модель обчислює, наскільки близько точки знаходяться до центрів кластерів.

3. Як визначити оптимальну кількість кластерів моделі?

Оптимальну кількість кластерів можна визначити кількома методами.

1. Метод ліктя (Elbow Method)

- Будують графік:
  по осі X — кількість кластерів k
  по осі Y — inertia_

Обирають точку, де різке зменшення змінюється плавним спадом. Ця точка називається "лікоть".

2. Silhouette Score

Оцінює:
- наскільки об’єкт схожий на свій кластер
- наскільки відрізняється від інших кластерів

Значення:
від -1 до 1

3. Davies-Bouldin Index

Показує якість поділу кластерів.

Чим менше значення — тим краще.
